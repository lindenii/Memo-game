MATCH-MATCH GAME

This is a counter game where the object is to find pairs.
You can also choose card`s skirt and game difficult.

HOW TO PLAY:

  1. When the game begins, all pictures are hidden.
  2. Select 2 cards to try to match the picture.
  3. If you match the pictures you can go again.
  4. If they dont`t match the computer turn them.
  5. The player that finds all pairs wins!

Choose a level!!!

You are the winner!!!
